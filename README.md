# personal_expenses

### Features:-
    1- Provider state management solution.
    2- Sqflite database,
    3- Datetime picker,
    4- Firebase auth.
    5- Facebook sign_in.
    6- Google signIn.
    7- Image picker.
    8- Email validator.
    9- Path provider.
    10- intl package.
